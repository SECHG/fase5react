Selector de Atributo

Los selectores de atributos se utilizan para seleccionar todos los elementos correspondientes a un atributo específico o valor de atributo definido. Por ejemplo, a[href] seleccionará todos los enlaces, mientras que a[href*="google"] solo seleccionará todas las URL que contengan la palabra "google". Este selector tiene una sintaxis un poco compleja la cual veremos líneas abajo:

  • [attr] { propiedades de estilo }
  • [attr=value] { propiedades de estilo }
  • [attr~=value] { propiedades de estilo }
  • [attr|=value] { propiedades de estilo }
  • [attr^=value] { propiedades de estilo }
  • [attr$=value] { propiedades de estilo }
  • [attr*=value] { propiedades de estilo }
