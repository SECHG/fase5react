FlexBox - Diego Gaviria

Que es Flex Box?


Flex box es un módulo de diseño de CSS3 diseñado para mejorar la forma en que se realiza el diseño receptivo, evitando el uso de flotantes, escribiendo menos código y facilitando la colocación de elementos, incluso si no conoce su tamaño. Básicamente la idea de Flex box es poder cambiar de forma óptima el ancho, el alto y la posición de los elementos en función del espacio del que dispongamos.


Estructura de Flex Box


Contenedor Flex:


Es nuestro contenedor padre, y es él quién va a contener a todos los elementos hijos a los cuáles queremos posicionar.


Elemento Flex


Es nuestro elemento hijo, el cual utilizará todo el espacio disponible para ubicarse de acuerdo con las propiedades a las cuales este sometido junto a los demás Elementos-Flex contenidos.


Cada diseño Flex está compuesto por dos ejes:


  • Un eje principal que define el posicionamiento horizontal de los Elementos-Flex.
  • Un eje secundario que define el posicionamiento vertical de los Elementos-Flex.

El Contendor-Flex contiene direcciones que definen el origen y el fin del flujo de Elementos-Flex:


  • Inicio principal
  • Fin principal
  • Inicio secundario
  • Fin secundario

Propiedades


Flex Direction


Encargada de definir cual será el eje principal y secundario



Flex Wrap


Cuando no se puede ordenar en una única línea los elementos, esta propiedad se encarga de solucionar el problema


Justify Content


Cuando nombramos el eje principal de un padre este hereda a los hijos para dar direccionamiento




Align Items


Esta propiedad al contrario de alinear en el eje horizontal lo hace en el vertical
